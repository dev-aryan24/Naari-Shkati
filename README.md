# Naari-Shkati
AI ML Early Detection of Real Time Threats For Women Safety

📌 Naari Shakti – AI-Based Women Safety System

Naari Shakti is an intelligent women safety platform that leverages Artificial Intelligence and real-time monitoring to detect potential threats, send instant alerts, and ensure proactive protection in critical situations.

🚀 Features  
🔹 Real-time threat detection using AI/ML  
🔹 Instant SOS alert system for emergencies  
🔹 Live location tracking for quick assistance  
🔹 User-friendly and responsive interface  
🔹 Fast alert mechanism for immediate response

🎯 Objective  

The objective of this project is to enhance women's safety by utilizing modern technology to provide real-time monitoring, faster emergency alerts, and intelligent threat detection, ultimately empowering women with a reliable security solution.

⚙️ How It Works    
User interacts with the application interface   
System continuously monitors inputs (location / behavior / triggers)   
AI model analyzes data for potential threats   
If a threat is detected, an instant alert/SOS is triggered   
Emergency contacts or authorities are notified with location details
